Face Tracking AI Drone

A Python script for the DJI Tello Drone to detect faces and follow them. The code uses a custom-made PID controller to change the drone's pitch, roll, and yaw to center around detected face using OpenCV face detection. The script uses DjiTelloPy, OpenCV, and PyGame.

Database Utilized Flashcard Application

This has been an ongoing project for me that I started in my second semester. The point of the project is to reinforce the theoretical concepts that I have learned in Data Structures and incorporating databases. The project is supposed to incorporate the concept of spaced repetition to make memorization easier. The project is written in Java, with the GUI being made with JavaFX. The flashcards, quizzes, and user's statistics are all stored in a Derby Database with the plan to switching to SQLite.

Arduino Addressable LED Lights

My first project using an Arduino, breadboard, various electronics, and an ESP32. The lights are W2812b fully addressable LEDs that can be controlled with microcontrollers. At first, I had the lights controlled through an arduino and ran the FastLED library to control them. Later, I switched the Arduino out for an ESP32 so that I can control the lights from my phone.

Rock Paper Scissors Game

My first attempt at making a website using HTML, JavaScript, and CSS. I made a simple rock, paper, scissors game. The user plays against the computer that randomly chooses an option. The game has animation on clicks and ends and resets when someone wins.

Music Recommender System

A Python Music Recommender Script that I made for my CS115 - Intro to Programming class. The system uses a data mining algorithm called collaborative filtering (CF). The script saves user's names and their artist recommendations into a text file and gives recommendations based on undiscovered similar artists from other users.

Unity Pong Recreation

My first attempt at making a Unity Game using C#. It is a recreation of Pong. The user is able to play against another person or against an AI with different difficulty levels.
